== Instalación ==

= Guía de instalación =

1. Regístrate en <a href="https://www.facturaone.com/registro" target="_blank">FacturaOne.com</a> y obten tu clave API desde Ajustes -> Catalogo -> WooCommerce.
2. Instala y activa el plugin de FacturaOne para tu WooCoommerce.
3. Entra en el panel de control de WordPress y ve a WooCommerce -> FacturaOne -> Ajustes y configuralo con tu clave API y tu email.
4. Felicidades!!! Ahora ya puedes gestionar tu Woocommerce desde ERP FacturaOne.
<a href="https://www.facturaone.com/contacto/" target="_blank">Si tienes cualquier duda ponte en contacto con nosotros</a>

= Requerimiento Minimo =

* WordPress 4.7 o superior
* PHP version 5.4 o superior
* MySQL version 5.0 o superior
* WooCommerce

= Recomendamos que tu host soporte: =

* PHP version 7.0 o superior
* MySQL version 5.6 o superior
* WordPress Memory limit of 64 MB o superior (128 MB o superior)



== Frequently Asked Questions ==

**¿Necesito tener una cuenta en FacturaOne para usarlo?**

Si, es necesario tener la cuenta para poder vincular tu APIKEY al plugin

**¿Necesito tener WooCommerce?**

Si, la principal caracteristica de este plugin es el uso de importación de informacion a WooCommerce.

**¿Puedo introducir productos desde la APP y que aparezcan en mi WooCommerce?**

Si, todos los cambios que se realize desde nuestra APP FacturaOne, se veran dentro de nuestro WooCommerce.

**¿Puedo tener tarifas de precio diferente por cliente?**

Los clientes que entren en web normalmente veran la tarifa WEB, pero puede establecer 5 tarifas, y asignarlas a los clientes que usaran estas tarifas.

**¿Puedo actualizar la información de mi tienda WooCommerce desde el ERP FacturaOne**

Si, cuando vinculas el WooCommerce a través del api puedes actualizar los productos, tarifas, clientes, categorias.
